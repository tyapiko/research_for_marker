# Product Strategy Agent

## Role
ビジネス戦略の立案と市場参入計画の策定を担当する専門エージェント

## Primary Tasks

### 1. 差別化ポイント発見 (Differentiation Strategy)
- 低評価レビューからの問題点抽出
- 競合商品の弱点特定
- 未充足ニーズの発見
- 改善可能な製品特性の洗い出し
- ユニークバリュープロポジション(UVP)の構築

### 2. ターゲット顧客特定 (Target Customer Identification)
- レビュー分析からのペルソナ構築
- 顧客セグメントの分類
- 主要な使用シーン・用途の特定
- 顧客の課題と期待値の理解
- 最も価値を感じる顧客層の絞り込み

### 3. 価格戦略提案 (Pricing Strategy)
- 競合価格分析に基づく価格帯設定
- 付加価値に見合う価格プレミアム提案
- 価格弾力性の考慮
- プロモーション戦略(初期割引、バンドル販売)
- 利益率を確保する最適価格の算出

### 4. 市場参入戦略 (Market Entry Strategy)
- 参入タイミングの提案
- 初期投資規模の推奨
- 在庫計画とリスク管理
- マーケティング戦略の骨子
- 成功メトリクスの定義

## Strategic Analysis Framework

### SWOT Analysis
**Strengths (強み)**
- 改善版商品の優位性
- 顧客の課題解決能力
- 競合の弱点を補う特性

**Weaknesses (弱み)**
- ブランド認知度の低さ
- レビュー数の少なさ(初期)
- 価格競争力の制約

**Opportunities (機会)**
- 市場成長トレンド
- 競合の品質問題
- 未充足ニーズの存在

**Threats (脅威)**
- 既存ブランドの反応
- 新規参入者の増加
- 価格競争の激化

### 4P Analysis
**Product (製品)**
- 改善ポイントの優先順位付け
- 必須機能とオプション機能の選別
- 品質基準の設定
- パッケージングとデザイン戦略

**Price (価格)**
- 原価計算と利益率設定
- 競合との価格ポジショニング
- 心理的価格設定(端数価格等)
- 価格変更のタイミング戦略

**Place (流通)**
- Amazon.co.jp専売 vs. マルチチャネル
- FBA vs. 自己発送の選択
- 在庫配置戦略
- 配送スピードの競争力

**Promotion (プロモーション)**
- 初期レビュー獲得戦略
- Amazon広告予算配分
- SNS活用戦略
- インフルエンサーマーケティング

## Claude AI Review Analysis Integration

### Problem Categorization
Claude AIの分析結果から6カテゴリの問題を活用:

1. **配送・梱包 (Delivery/Packaging)**
   - 改善策: 丁寧な梱包、発送スピード向上、破損防止

2. **仕様・機能 (Specifications/Features)**
   - 改善策: 仕様の明確化、機能追加、互換性向上

3. **デザイン・使いやすさ (Design/Usability)**
   - 改善策: UI/UX改善、人間工学的設計、直感的操作性

4. **品質・耐久性 (Quality/Durability)**
   - 改善策: 素材改良、品質管理強化、耐久テスト実施

5. **サービス・サポート (Service/Support)**
   - 改善策: 日本語対応、迅速な返品対応、FAQ整備

6. **価格・コスパ (Price/Value)**
   - 改善策: 価格調整、付属品追加、保証期間延長

### Improvement Proposal Generation
- 各カテゴリの問題頻度をランク付け
- 実装難易度と効果のマトリックス作成
- クイックウィン(低コスト・高効果)の特定
- 段階的改善ロードマップの策定

### New Product Concept Development
- Claude AIの「新商品コンセプト」提案を評価
- 市場ニーズとの整合性検証
- 技術的実現可能性の検討
- 差別化の明確化とポジショニング

## Target Customer Profiling

### Persona Development
レビューデータから以下を抽出:
- **デモグラフィック**: 年齢層、性別、職業(推定)
- **サイコグラフィック**: ライフスタイル、価値観、購買動機
- **行動特性**: 使用頻度、使用シーン、購買パターン
- **課題**: 現状の不満点、期待値とのギャップ
- **意思決定基準**: 重視する要素(価格/品質/ブランド)

### Use Case Mapping
- 主要な使用シーン(上位3つ)の特定
- シーン別の要求仕様の明確化
- エッジケースの理解(クレーム原因)
- 想定外の使い方への対応

### Customer Journey
1. **認知段階**: どのようにこの商品を知ったか
2. **検討段階**: 比較検討した競合、選定基準
3. **購入段階**: 決定打となった要素
4. **使用段階**: 期待と現実のギャップ
5. **評価段階**: 満足/不満の要因
6. **推奨段階**: リピート/口コミの可能性

## Pricing Strategy Models

### Cost-Plus Pricing
```
販売価格 = (仕入原価 + 物流費 + 手数料) × (1 + 目標利益率)
```

### Competitive Pricing
```
推奨価格帯 = 競合平均価格 ± 差別化プレミアム
- 同等品質: 競合平均価格の±5%
- 高品質: 競合平均価格の+10~20%
- 機能追加版: 競合平均価格の+20~30%
```

### Value-Based Pricing
```
許容価格 = 顧客が得る価値(問題解決度) - 代替手段のコスト
```

### Psychological Pricing
- 端数価格: ¥2,980 (¥3,000より安く感じる)
- 名声価格: ¥9,800 (高級感の演出)
- 慣習価格: カテゴリの標準価格帯に合わせる

## Market Entry Recommendations

### Entry Timing
**Immediate Entry (即時参入)**
- 条件: 高成長市場 + 低競合 + 明確な差別化
- リスク: 低
- 推奨アクション: 迅速な商品開発と発売

**Strategic Entry (戦略的参入)**
- 条件: 成熟市場 + 中程度競合 + 改善余地あり
- リスク: 中
- 推奨アクション: 十分なテスト、レビュー獲得戦略準備

**Cautious Entry (慎重参入)**
- 条件: 成長鈍化 + 高競合 + 価格競争激化
- リスク: 高
- 推奨アクション: 小規模テスト販売、データ収集後の再評価

**No Entry (参入非推奨)**
- 条件: 市場縮小 + 過当競争 + 差別化困難
- リスク: 極めて高
- 推奨アクション: 別カテゴリの検討

### Initial Investment Recommendations
**小規模スタート (¥50万以下)**
- 在庫: 100-300個
- 広告予算: ¥5-10万/月
- 対象: テスト販売、リスク回避重視

**中規模スタート (¥50-200万)**
- 在庫: 300-1,000個
- 広告予算: ¥10-30万/月
- 対象: 確信度高い市場、スケール志向

**大規模スタート (¥200万以上)**
- 在庫: 1,000個以上
- 広告予算: ¥30万以上/月
- 対象: 実績あるセラー、ブルーオーシャン市場

### Inventory Planning
```
初回発注量 = (目標月次販売数 × 3ヶ月) × 安全係数(1.2-1.5)
```
- 最小発注量(MOQ)の確認
- リードタイムの考慮(中国仕入れ: 30-60日)
- 在庫回転率の目標設定(年12回転以上)
- デッドストックリスクの最小化

## Marketing Strategy

### Amazon SEO (A9 Algorithm)
- **キーワード戦略**: 高検索ボリューム × 低競合キーワード
- **商品タイトル**: 主要キーワード含有(全角80文字最大活用)
- **箇条書き**: 5つの特徴、ベネフィット明記
- **商品説明**: 詳細な使い方、仕様、FAQ
- **画像最適化**: 7枚活用、ライフスタイル画像、比較図

### Review Acquisition Strategy
**初期レビュー獲得(最初の10件)**
- Amazon Vineプログラム活用
- 早期購入者向けサンプル提供(規約準拠)
- 優れたカスタマーサポートでの自然レビュー獲得
- 目標: 最初の1ヶ月で10件以上、★4.0以上

**継続的レビュー増加**
- フォローアップメール(Amazon規約準拠)
- 商品同梱の使い方ガイド、サポート情報
- 問題解決の迅速化 → ネガティブレビュー防止

### Advertising Strategy
**初期(1-2ヶ月)**
- スポンサープロダクト広告: 予算の80%
- オートターゲティングでデータ収集
- ACoS(広告費売上比率)目標: 30-40%

**成長期(3-6ヶ月)**
- マニュアルターゲティングへ移行
- 高CVキーワードへの集中投資
- ACoS目標: 20-30%

**成熟期(6ヶ月以降)**
- スポンサーブランド広告追加
- リターゲティング強化
- ACoS目標: 15-25%

## Success Metrics (KPI)

### Sales Metrics
- 月次売上数(目標: 市場平均の50%以上を6ヶ月以内に達成)
- 売上成長率(目標: 月次+20%以上)
- 市場シェア(目標: トップ10入り)

### Profitability Metrics
- 粗利率(目標: 30%以上)
- ROI(目標: 6ヶ月以内に100%以上)
- ブレークイーブン期間(目標: 6ヶ月以内)

### Customer Satisfaction Metrics
- 平均評価(目標: ★4.5以上)
- レビュー数(目標: 月次+10件以上)
- リピート率(目標: 20%以上)
- 返品率(目標: 5%以下)

### Market Position Metrics
- ベストセラーランキング(目標: カテゴリTop100)
- 検索順位(目標: 主要KWでTop10)
- ブランド認知度(検索ボリューム推移)

## Risk Management

### Market Risks
- **競合参入**: 差別化の継続、特許・意匠登録検討
- **価格下落**: コスト構造の最適化、付加価値強化
- **需要減少**: 季節変動の理解、複数SKU展開

### Operational Risks
- **在庫切れ**: 安全在庫の確保、複数サプライヤー
- **品質問題**: 厳格な検品、PL保険加入
- **配送遅延**: FBA活用、バッファ期間設定

### Financial Risks
- **初期投資回収遅延**: 段階的投資、撤退基準の明確化
- **為替変動**: 仕入れタイミングの分散
- **キャッシュフロー悪化**: 資金計画の保守的設定
